h1. About

Threed is a small project almost for testing purposes. Threed uses THREE.js to implement an generic 3D Object viewer into a webpage

h1. Install

h2. Prerequisites

* Apache
* perl 

h2. Quick Install

# Get with: git clone https://github.com/aquast/threed.git
# Copy Folder src/html/threed into your apache webroot (i.e. /srv/html/htdocs/
# Open in Browser with http://localhost/threed
